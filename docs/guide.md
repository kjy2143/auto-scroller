# 자동 스크롤러 개발/배포/설치 가이드

## 1. 개발 환경 준비

- Node.js 18 이상 권장
- 의존성 설치:
  ```bash
  npm install
  ```

## 2. 개발 서버 실행

```bash
npm run dev
```
- http://localhost:5173 에서 개발 서버 확인

## 3. 빌드 및 북마클릿 변환

1. 빌드 실행 및 북마클릿 코드 생성
   ```bash
   npm run build
   ```
   - 빌드 결과물: dist/auto-scroller.min.js
   - 북마클릿 코드: dist/bookmarklet.txt 파일에 자동 생성됨

## 4. 북마클릿 설치 방법

1. dist/bookmarklet.txt 파일을 열어 전체 코드를 복사
2. 브라우저 즐겨찾기(북마크) 추가
3. URL 입력란에 복사한 코드를 붙여넣고 저장
4. 원하는 웹페이지에서 해당 북마크 클릭 시 자동 스크롤 동작

## 5. 주요 기능 및 단축키

- 우측 하단 ▶️/⏸️ 버튼으로 자동 스크롤 시작/정지
- S 키로도 토글 가능(입력창/textarea 등에서는 동작 안 함)
- 페이지 하단 도달 시 자동 정지 및 알림

## 6. 접근성 및 유지보수

- 버튼에 aria-label, 키보드 접근성, 포커스 표시 적용
- 주요 함수/로직에 한글 주석 보강
